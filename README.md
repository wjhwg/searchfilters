# searchfilters

## Build Setup

## 涵盖以下几个功能：

### 1.输入框模糊查询

### 2.带有滚动显示区，显示模糊查询到的内容

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
